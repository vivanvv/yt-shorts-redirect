Privacy Policy

For the YouTube Shorts Redirect browser extension.

Summary

YouTube Shorts Redirect does not collect, store, transmit, sell, or share personal data.

How the extension works

The extension runs locally in the browser. When a user opens a YouTube Shorts URL such as https://www.youtube.com/shorts/<videoId>, the extension redirects that page to the standard YouTube watch URL format https://www.youtube.com/watch?v=<videoId>.

Data collection and use

The extension does not collect or retain personal information, account information, browsing history for external use, authentication data, financial information, health information, personal communications, location data, or user-generated content.

The extension only reads the current YouTube page URL locally in order to determine whether it should perform the redirect.

Data sharing

No user data is sold, transferred, or shared with third parties.

Remote code and tracking

The extension does not use remote code, analytics, advertising, or third-party tracking.

Changes

If the extension's data practices change in the future, this policy will be updated before those changes are released.
