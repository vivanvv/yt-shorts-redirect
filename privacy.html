Privacy Policy

For the YouTube Shorts Redirect browser extension.

What the extension does

YouTube Shorts Redirect runs locally in the browser and redirects https://www.youtube.com/shorts/<videoId> URLs to https://www.youtube.com/watch?v=<videoId>.

Data collection

This extension does not collect, store, transmit, sell, or share personal data.

Data use

The extension only reads the current YouTube URL locally to determine whether a redirect should happen.

Data sharing

No user data is sold, transferred, or shared with third parties.

Remote code

The extension does not use remote code.
