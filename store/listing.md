# Chrome Web Store Listing Draft

## Name

YouTube Shorts Redirect

## Summary

Redirect YouTube Shorts links to the standard YouTube watch page.

## Description

YouTube Shorts Redirect automatically converts YouTube Shorts URLs into normal watch URLs.

This extension listens for YouTube navigation changes and redirects:

- `https://www.youtube.com/shorts/<videoId>`

to:

- `https://www.youtube.com/watch?v=<videoId>`

It keeps existing query parameters and URL fragments where possible.

## Privacy

Suggested dashboard answers, assuming the current code stays unchanged:

- Data collection: No
- Authentication: No
- Remote code: No
- Single purpose: Redirect YouTube Shorts URLs to standard watch URLs
